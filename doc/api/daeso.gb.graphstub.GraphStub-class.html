Class GraphStub

source code

object --+
         |
        GraphStub

A GraphStub is a stub for a Daeso graph instance. Its only content is the graph's id. It used in the PGCParser class and SparseGraphBank class to prevent parsing unaligned graphs (creating sparse graph bank) or to avoid parsing alltogether (reading a corpus without actually opening graph banks). See those classes for details.

A GraphStub object maintains a list of GraphPair objects, called "clients" (in self.clients), which use the stub as their source and/or target graph. This enables us to replace the stub by a real graph later on, using the method "replace_stub".

GraphStub object are not supposed to be reused.
