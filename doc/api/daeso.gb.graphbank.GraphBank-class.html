__eq__(self, other)
(Equality operator)

source code 

self == other

Two graphbanks are equal if and only if their file paths point to the same file. Unless the paths are string identical, the OS is invoked the check if the paths point to the same existing file.

(Two graphbanks in different files may in fact be equal, but for reasons of simplicity and efficiency no attempt is made to analyse their contents.)
